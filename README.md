# Node.js + React를 사용한 익스프레스 웹 SNS 사이트 제작 프로젝트

>
> 넌적스 템플릿엔진을 사용하여 데이터를 동적으로 보여줌
>
> 웹 서버는 익스프레스 사용
>
> About 페이지로 이동했을 때 화면을 React로 구현하였음
